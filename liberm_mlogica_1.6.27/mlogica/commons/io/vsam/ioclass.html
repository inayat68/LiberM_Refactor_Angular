    Customized VSAM structures

    VSAM files can be mapped to custom user-defined storage structures, with full control over data management delegated to the user.

    This approach allows storing the content of a single VSAM file across one or more database tables, potentially following a structured data model—for example, a table with individual columns representing each field of the original record layout.

    To enable this functionality, the user must supply a custom I/O class, i.e., a Java class that implements the required low-level I/O operations.

    The I/O class must be provided for the VSAM cluster definition. AIX and PATHs will use the same I/O class as the related cluster.

    Cluster definition

    The I/O class must be specified in the cluster definition, either:

    • providing the IOCLASS parameter in the NIBCAMS1 DEFINE CLUSTER command, or
    • by using the --io-class option in the nibvsam command line tool.

    Example with NIBCAMS1:

    Groovy
    exec "DEFCLUST", pgm: "IDCAMS", {
    dd "SYSPRINT", sysout: "*"
    // Instream content is expanded below
    dd "SYSIN", instream: [
            " DEFINE   CLUSTER (NAME(TEST.IOCLASS.USRSEC.KSDS01.KSDS)     -",
            "                    KEYS(8,0)                                 -",
            "                    RECORDSIZE(80,80)                         -",
            "                    REUSE                                     -",
            "                    INDEXED                                   -",
            "                    IOCLASS(org.sample.batch.CardDemoUserIO)  -",
            "                    TRACKS(45,15)                             -",
            "                    FREESPACE(10,15)                          -",
            "                    CISZ(8192))                               -",
            "          DATA    (NAME(TEST.IOCLASS.USRSEC.KSDS01.KSDS.DAT)) -",
            "          INDEX   (NAME(TEST.IOCLASS.USRSEC.KSDS01.KSDS.IDX))"
    ]
}

    Example with nibvsam command line tool:

    Bash
    nibvsam define cluster --io-class=com.nib.regtest.programs.CardDemoUserIOClone \
        --ksds \
        --lrecl=80,80 \
        --key=0,8 \
        TEST.IOCLASS.USRSEC.KSDS01.KSDS

    Class loading

    The I/O class must be made available in the default classpath of the JVM running the LIBER*M framework.

    IVsamApi Interface Summary

    All I/O classes must implement the com.nib.commons.io.vsam.IVsamApi interface.

    As an interface, all methods defined in IVsamApi must be implemented, even if some are not invoked at runtime. For instance, an I/O class designed for KSDS files is required to implement the entire interface; however, only the KSDS-specific methods will be utilized in practice. Methods related to other formats—such as those using RRN—will not be called and can therefore be implemented as no-ops or with empty bodies

    The table below provides a summary of the expected behavior for each method.

    Method Description
    defineCluster(FileDescriptor fd) Defines a new VSAM cluster.
    defineAix(FileDescriptor fd, KsdsKey kf) Defines an Alternate Index (AIX).
    deleteCluster(FileDescriptor fd) Deletes a VSAM cluster.
    deleteAix(FileDescriptor fd) Deletes an Alternate Index.
    alterCluster(FileDescriptor ofd, FileDescriptor nfd) Alters an existing cluster with a new configuration.
    rebuildIndexes(FileDescriptor fd) Rebuilds indexes for a VSAM file.
    open(OpenMode openMode, LockMode lockMode) Opens the VSAM file with specific modes.
    close() Closes the file.
    count(FileDescriptor fd) Returns number of records in the file.
    getNextRbaToAppend() Retrieves the next available RBA.
    read(KsdsKey keyDef, byte[] keyValue, byte[] recordBuffer, KeyIs mode, boolean update) Reads a record by key (KSDS)
    read(Rrn rrn, byte[] recordBuffer, KeyIs mode, boolean update) Reads a record by RRN (RRDS)
    read(Rba rba, byte[] recordBuffer, KeyIs mode, boolean update) Reads a record by RBA (ESDS)
    write(byte[] record, int reclen) Writes a record.
    write(Rrn rrn, byte[] record, int reclen) Writes a record at a specific RRN.
    write(Rba rba, byte[] record, int reclen) Writes a record at a specific RBA.
    rewrite(KsdsKey keyDef, byte[] key, byte[] record, int reclen) Rewrites a record by key.
    rewrite(Rrn rrn, byte[] record, int reclen) Rewrites a record by RRN.
    rewrite(Rba rba, byte[] record, int reclen) Rewrites a record by RBA.
    delete(KsdsKey keyDef, byte[] keyValue, boolean generic) Deletes one or more records by key.
    void delete(Rrn rrn) Deletes a record by RRN.
    exportData(SequentialFile target, boolean append) Exports file content to a sequential file.
    importData(SequentialFile source, int batchSize) Imports records from a sequential file.
    retrieveLastRrnWritten() Returns the last written Relative Record Number (RRN).

    AbstractIOClass facility

    To simplify the development of custom I/O classes and reduce boilerplate code, the framework provides the com.nib.commons.io.vsam.AbstractIOClass base class.

    This abstract class offers default implementations for all methods defined in the IVsamApi interface, allowing developers to override only the methods relevant to their specific use case. This improves both readability and maintainability of I/O class implementations.

    Each default implementation in AbstractIOClass throws an exception at runtime. This design ensures that if a method is unintentionally left unimplemented but invoked during execution, it will fail explicitly—helping developers quickly identify and address missing overrides.

    Below is an example of a custom I/O class extending AbstractIOClass, focused on persisting VSAM records into a relational database.

    Java
    package com.nib.regtest.programs;

import com.nib.commons.io.*;
import com.nib.commons.io.Record;
import com.nib.commons.io.exceptions.EndOfFileException;
import com.nib.commons.io.exceptions.FileIOException;
import com.nib.commons.io.vsam.*;
import com.nib.commons.storage.Memory;
import com.nib.commons.utils.NotImplementedException;

import java.sql.*;
import java.util.Arrays;

public class CardDemoUserIO extends AbstractIOClass {

    public static final String SQL_TRUNCATE = "truncate table regtest_carddemo_users";
    public static final String SELECT_1_PRIMARY_KEY = "select * from regtest_carddemo_users where USER_ID %s ? order by USER_ID %s limit 1 %s";
    public static final String SELECT_1_AIX_1 = "select * from regtest_carddemo_users where LAST_NAME %s ? order by LAST_NAME %s limit 1 %s";

    public CardDemoUserIO(AbstractFile file, Connection conn) {
        super(file, conn);
    }

    static final String SQL_INSERT = "insert into regtest_carddemo_users values (?, ?, ?, ?, ?)";
    static final String SQL_CREATE_CLUSTER = """
            create table regtest_carddemo_users
            (
                user_id CHAR(8) not null primary key,
                first_name varchar(20),
                last_name varchar(20),
                pass char(8) not null,
                user_type char(1)
            );
            """;
    static final String SQL_CREATE_AIX = """
            create index regtest_carddemo_users_surname
                      on regtest_carddemo_users (last_name)
            """;
    static final String SQL_UPDATE = """
            update regtest_carddemo_users set 
                user_id = ?,
                first_name = ?,
                last_name = ?,
                pass = ?,
                user_type = ?
            where user_id = ?
            """;

    static final String SQL_DELETE_EQUAL = "delete from regtest_carddemo_users where user_id = ?";
    static final String SQL_DELETE_GENERIC = "delete from regtest_carddemo_users where user_id >= ?";
    static final String SQL_SELECT_ALL = "select * from regtest_carddemo_users order by user_id";

    @Override
    public void deleteCluster(FileDescriptor fd) throws FileIOException {
        try (Statement stm = connection.createStatement()) {
            String createSql = "drop table regtest_carddemo_users";
            stm.execute(createSql);
        } catch (SQLException sqle) {
            throw new FileIOException(fd, FileStatus.ERROR, "error deleting table: " + sqle.getMessage(), sqle);
        } catch (Exception e) {
            throw new FileIOException(fd, FileStatus.ERROR, "error deleting table: " + e.getMessage(), e);
        }
    }

    @Override
    public void rebuildIndexes(FileDescriptor fd) {
    }

    @Override
    public void defineCluster(FileDescriptor fd) throws FileIOException {
        try (Statement stm = connection.createStatement()) {
            stm.execute(SQL_CREATE_CLUSTER);
        } catch (Exception e) {
            throw new FileIOException(fd, FileStatus.ERROR, "Error creating table: " + e.getMessage(), e);
        }
    }

    @Override
    public void defineAix(FileDescriptor fd, KsdsKey keyDef) throws FileIOException {
        String idx = keyDef.getName();
        String sql = SQL_CREATE_AIX;
        try (Statement stm = connection.createStatement()) {
            stm.execute(sql);
        } catch (Exception e) {
            throw new FileIOException(fd, FileStatus.ERROR, "Error creating AIX: " + e.getMessage(), e);
        }
    }

    @Override
    public void open(OpenMode openMode, LockMode lockMode) throws FileIOException {
        if (!tableExists("regtest_carddemo_users")) {
            throw new TableNotFoundException(file);
        }
        if (openMode == OpenMode.Output) {
            truncate();
        }
    }

    boolean tableExists(String table) throws FileIOException {
        String sql = String.format("""
                        SELECT COUNT(*) FROM INFORMATION_SCHEMA.TABLES 
                            WHERE TABLE_SCHEMA=CURRENT_SCHEMA AND UPPER(TABLE_NAME)='%s'
                        """,
                table.toUpperCase());
        try (Statement stm = connection.createStatement()) {
            var q = stm.executeQuery(sql);
            if (q.next()) {
                return q.getInt(1) > 0;
            }
        } catch (SQLException e) {
            throw new FileIOException(file, FileStatus.ERROR, "Error checking table existance: " + e.getMessage(), e);
        }
        return false;
    }

    protected void truncate() {
        try (Statement stm = connection.createStatement()) {
            String sql = SQL_TRUNCATE;
            log.debug(sql);
            stm.execute(sql);
        } catch (Exception e) {
            throw new FileIOException(file, FileStatus.ERROR, "Error truncating table: " + e.getMessage(), e);
        }
    }

    @Override
    public int read(KsdsKey keyDef, byte[] keyValue, byte[] recordBuffer, KeyIs mode, boolean update) {
        String sqlBase = switch (keyDef.getOffset()) {
            case 0 -> SELECT_1_PRIMARY_KEY;
            case 28 -> SELECT_1_AIX_1;
            default -> throw new FileIOException(file, FileStatus.ERROR, "Key not supported");
        };
        String order = switch (mode) {
            case LTEQ, LESS -> "DESC";
            default -> "ASC";
        };
        String sql = String.format(sqlBase,
                mode.operator,
                order,
                update ? "FOR UPDATE" : ""
        );
        String key = new String(keyValue);
        try (PreparedStatement statement = connection.prepareStatement(sql)) {
            statement.setString(1, key.trim());
            try (ResultSet rs = statement.executeQuery()) {
                if (rs.next()) {
                    CardDemoUser user = new CardDemoUser();
                    user.allocate();
                    // filling record with DB values
                    user.secUsrId.setValue(rs.getString("user_id"));
                    user.secUsrFname.setValue(rs.getString("first_name"));
                    user.secUsrLname.setValue(rs.getString("last_name"));
                    user.secUsrPwd.setValue(rs.getString("pass"));
                    user.secUsrType.setValue(rs.getString("user_type"));
                    user.secUsrFiller.lowValues();
                    // preparing last key read
                    Arrays.fill(keyValue, (byte) 0);
                    byte[] kv = switch (keyDef.getOffset()) {
                        case 0 -> user.secUsrId.getBytes();
                        case 28 -> user.secUsrLname.getBytes();
                        default -> throw new NotImplementedException();
                    };
                    System.arraycopy(kv, 0, keyValue, 0, user.secUsrId.length());
                    // preparing record
                    Arrays.fill(recordBuffer, (byte) 0);
                    System.arraycopy(user.getBytes(), 0, recordBuffer, 0, user.length());
                    return user.length();
                } else {
                    throw new KeyNotFoundException(file);
                }
            }
        } catch (FileIOException fioe) {
            throw fioe;
        } catch (Exception e) {
            log.error("IOCLASS READ", e);
            throw new FileIOException(file, FileStatus.ERROR, e.getMessage(), e);
        }
    }

    @Override
    public void write(byte[] record, int reclen) throws FileIOException {
        try (PreparedStatement statement = connection.prepareStatement(SQL_INSERT)) {
            CardDemoUser user = new CardDemoUser();
            if (record.length < user.length() || reclen < user.length()) {
                throw new FileIOException(file, FileStatus.ERROR, "Illegal input record length");
            }
            user.allocate(new Memory(record));
            statement.setString(1, user.secUsrId.getValue());
            statement.setString(2, user.secUsrFname.getValue().trim());
            statement.setString(3, user.secUsrLname.getValue().trim());
            statement.setString(4, user.secUsrPwd.getValue());
            statement.setString(5, user.secUsrType.getValue());
            int rowsInserted = statement.executeUpdate();
            if (rowsInserted != 1) {
                throw new FileIOException(file, FileStatus.ERROR, "Cannot insert into io table regtest_carddemo_users");
            }
        } catch (SQLException sqle) {
            checkVsamSqlException(IOOperation.WRITE, sqle);
        } catch (FileIOException fioe) {
            throw fioe;
        } catch (Exception e) {
            throw new FileIOException(file, FileStatus.ERROR, "Error writing storage: " + e.getMessage(), e);
        }
    }

    @Override
    public void rewrite(KsdsKey keyDef, byte[] keyValue, byte[] record, int reclen) {
        String keyName = keyDef.getName();
        String sql = SQL_UPDATE;
        try (PreparedStatement statement = connection.prepareStatement(sql)) {
            CardDemoUser user = new CardDemoUser();
            if (record.length < user.length() || reclen < user.length()) {
                throw new FileIOException(file, FileStatus.ERROR, "Illegal input record length");
            }
            user.allocate(new Memory(record));
            statement.setString(1, user.secUsrId.getValue());
            statement.setString(2, user.secUsrFname.getValue().trim());
            statement.setString(3, user.secUsrLname.getValue().trim());
            statement.setString(4, user.secUsrPwd.getValue());
            statement.setString(5, user.secUsrType.getValue());
            statement.setString(6, user.secUsrId.getValue());
            int rowsUpdated = statement.executeUpdate();
            switch (rowsUpdated) {
                case 0 -> throw new KeyNotFoundException(file);
                case 1 -> {
                }
                default -> throw new FileIOException(file, FileStatus.ERROR, "multiple rows selected");
            }
        } catch (SQLException sqle) {
            checkVsamSqlException(IOOperation.REWRITE, sqle);
        } catch (FileIOException fioe) {
            throw fioe;
        } catch (Exception e) {
            throw new FileIOException(file, FileStatus.ERROR, "Error rewriting storage: " + e.getMessage(), e);
        }
    }

    public int delete(KsdsKey keyDef, byte[] keyValue, boolean generic) throws FileIOException {
        String keyName = keyDef.getName();
        String sql = generic ? SQL_DELETE_GENERIC : SQL_DELETE_EQUAL;
        try (PreparedStatement statement = connection.prepareStatement(sql)) {
            CardDemoUser user = new CardDemoUser();
            String userid = new String(keyValue).trim();
            statement.setString(1, userid);
            int rows = statement.executeUpdate();
            if (rows == 0) {
                throw new KeyNotFoundException(file);
            }
            return rows;
        } catch (FileIOException fioe) {
            throw fioe;
        } catch (Exception e) {
            throw new FileIOException(file, FileStatus.ERROR, "Error deleting storage: " + e.getMessage(), e);
        }
    }

    @Override
    public void close() throws FileIOException {
    }

    @Override
    public long importData(SequentialFile infile, int batchSize) throws FileIOException {
        long count = 0;
        boolean ee = infile.isExceptionsEnabled();
        boolean ac = false;
        try {
            ac = connection.getAutoCommit();
            connection.setAutoCommit(false);
        } catch (SQLException e) {
            throw new RuntimeException(e);
        }
        infile.setExceptionsEnabled(true);
        infile.open(OpenMode.Input);

        byte[] rec = infile.getRecord().getBytes();
        CardDemoUser user = new CardDemoUser();
        if (rec.length < user.length()) {
            throw new FileIOException(file, FileStatus.ERROR, "Illegal input record length");
        }
        user.allocate(new Memory(rec));
        try (PreparedStatement statement = connection.prepareStatement(SQL_INSERT)) {
            try {
                while (true) {
                    infile.read();
                    statement.setString(1, user.secUsrId.getValue());
                    statement.setString(2, user.secUsrFname.getValue().trim());
                    statement.setString(3, user.secUsrLname.getValue().trim());
                    statement.setString(4, user.secUsrPwd.getValue());
                    statement.setString(5, user.secUsrType.getValue());
                    statement.addBatch();
                    if (++count % batchSize == 0) { // Execute batch every 10K rows
                        statement.executeBatch();
                        connection.commit(); // Commit every batch
                    }
                }
            } catch (EndOfFileException eof) {
                statement.executeBatch();
                connection.commit();
            }
            return count;
        } catch (FileIOException fioe) {
            throw fioe;
        } catch (Exception e) {
            throw new FileIOException(file, FileStatus.ERROR, "Error importing data: " + e.getMessage(), e);
        } finally {
            if (infile.isOpen()) infile.close();
            infile.setExceptionsEnabled(ee);
            try {
                connection.setAutoCommit(ac);
            } catch (SQLException e) {
                throw new RuntimeException(e);
            }
        }
    }

    @Override
    public long exportData(SequentialFile outfile, boolean append) throws FileIOException {
        long count = 0;
        Record outrec = outfile.getRecord();
        byte[] recordBuffer = outfile.getRecord().getBytes();
        outfile.open(append ? OpenMode.Append : OpenMode.Output);
        try (PreparedStatement statement = connection.prepareStatement(SQL_SELECT_ALL)) {
            try (ResultSet rs = statement.executeQuery()) {
                while (rs.next()) {
                    CardDemoUser user = new CardDemoUser();
                    user.allocate();
                    // filling record with DB values
                    user.secUsrId.setValue(rs.getString("user_id"));
                    user.secUsrFname.setValue(rs.getString("first_name"));
                    user.secUsrLname.setValue(rs.getString("last_name"));
                    user.secUsrPwd.setValue(rs.getString("pass"));
                    user.secUsrType.setValue(rs.getString("user_type"));
                    user.secUsrFiller.spaces(); // <-- I want the export file to be padded with spaces.
                    // preparing record
                    Arrays.fill(recordBuffer, (byte) 0);
                    System.arraycopy(user.getBytes(), 0, recordBuffer, 0, user.length());
                    outrec.setLength(user.length());
                    outfile.write();
                    count++;
                }
            }
        } catch (FileIOException fioe) {
            throw fioe;
        } catch (Exception e) {
            throw new FileIOException(file, FileStatus.ERROR, "Error exporting data", e);
        } finally {
            outfile.close();
        }
        return count;
    }
