    File Control (FCT)

    The FCT is the configuration file that defines the files (datasets) available in the LIBER*TP region.

    The FCT is a JSON file that contains an array of file resource objects.

    Each file object contains the following properties:

    Property Description
    id The file ID.
    group The file group.
    dsn The file dataset name.
    browsable The file is browsable.
    addable The file is addable.
    deletable The file is deletable.
    readable The file is readable.
    updatable The file is updatable.

    Example

    YAML
    {
  "FCT": [
    {
      "id": "ACCTDAT",
      "group": "CARDDEMO",
      "dsn": "AWS.M2.CARDDEMO.ACCTDATA.VSAM.KSDS"
    },
    {
      "id": "CARDAIX",
      "group": "CARDDEMO",

    },
    {
      "id": "CARDDAT",
      "group": "CARDDEMO",
      "dsn": "AWS.M2.CARDDEMO.CARDDATA.VSAM.KSDS"
    },
    {
      "id": "CCXREF",
      "group": "CARDDEMO",
      "dsn": "AWS.M2.CARDDEMO.CARDXREF.VSAM.KSDS"
    },
    {

      "dsn": "AWS.M2.CARDDEMO.CUSTDATA.VSAM.KSDS"
    },
    {
      "id": "CXACAIX",

    },

    },
    {
