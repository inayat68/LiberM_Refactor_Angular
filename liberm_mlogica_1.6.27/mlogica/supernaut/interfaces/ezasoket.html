    EZASOKET

    EZASOKET is a CICS interface that allows CICS programs to communicate with other CICS programs using a socket interface.

    Sockets are available by default in Java and .NET, so the migrated applications can use directly the native socket interfaces.

    However, CICS provides a socket listener, the EZALISTENER indeed, that can be used to start transactions via sockets. Using the GIVESOKET/TAKESOKET mechanism, such transactions can communicate with the client.

    LIBER*TP provides a TCP/IP server that replicates the EZALISTENER feature. This server can be used to start transactions and the latter can communicate with the client using the native socket interfaces.

    EZASOKET listener can be configured to use SSL.

    To configure one or more listeners, a tcp service configuration file must be specified in the supernaut configuration file:

    YAML
    supernaut:
  # ...
  tcp-services:
    config: 'file:./tcp-services.json'

    The listeners configuration file is a JSON file that contains an array of listener configurations. Each listener is defined by a JSON object as follows:

    Field Description
    name The name of the listener.
    address The address to listen to.
    port The port number to listen to.
    userid The user ID associated with the started transactions.
    ssl If true, uses SSL.
    keystore For SSL, path to SSL keystore, in PKCS12 format.
    keystorePassword For SSL, keystore password.
    requireClientAuth For SSL, if true client authorization is requested.

    Example:

    JSON
    {
  "TcpServices": [
    {
      "serviceType": "Ezalistener",
      "name": "LIST2323",
      "address": "localhost",
      "port": "2323",
      "userid": "EZASOKET"
    },
    {
      "serviceType": "Ezalistener",
      "name": "OTHER888",
      "address": "0.0.0.0",
      "port": "1888",
      "userid": "EZASOKET"
    },
    {
      "serviceType": "Ezalistener",
      "name": "GEEZER",
      "address": "localhost",
      "port": "2324",
      "userid": "EZASOKET",
      "ssl": true,
      "keystore": "keystore.p12",
      "keystorePassword": "lafavaelarava",
      "requireClientAuth": "false"
    }
  ]
}

    The LIBER*TP Ezalistener accepts data in the standard CICS format: TRANSACTION,USERDATA,START_MODE,HHMMSS:

    • TRANSACTION: TRANSID to start.
    • USERDATA: 32 bytes user data to pass to the transaction.
    • START_MODE: KC, IC or TD. Only KC or IC,000000 are supported.
    • HHMMSS: Time to start the transaction (ignored).

    As on CICS, it starts the transaction specified in the message providing a commarea matching the TCP-SOCKET-PARM structure, which is defined as follows in the class com.nib.supernaut.tcp.TcpSocketParm:

    Java
    public class TcpSocketParm extends NGroup {
    public final NInt32 giveTakeSocket = new NInt32();

    public static class ClientId extends NGroup {
        public final NInt32 lstnCidDomain = new NInt32();
        public final NChar lstnCidName = new NChar(8);
        public final NChar lstnCidTask = new NChar(8);
        public final NChar lstnCidRsvd = new NChar(20);
    }

    public final ClientId clientId = new ClientId();
    public final NChar clientInData = new NChar(35);
    public final NChar filler = new NChar(1);

    public static class SockAddrInParm extends NGroup {
        public final NInt32 sinFamily = new NInt32();
        public final NInt32 sinPort = new NInt32();
        public final NInt32 sinAddress = new NInt32();
        public final NChar sinZero = new NChar(8);
    }

    public final SockAddrInParm sockAddrInParm = new SockAddrInParm();
}

    Started transaction

    The started program can use the sockets passed by the listener using the EzaSocket class.

    Example:

    Java
    public class EZASRV01 extends AbstractOnlineProgram {
    public EZASRV01(Context rt) {
        super(rt);
    }

    class WorkingStorage extends NGroup {
        NInt16 len = new NInt16();
        TcpSocketParm tcpSockParm = new TcpSocketParm();
    }

    @ProgramStorage
    final WorkingStorage WS = new WorkingStorage();

    @Override
    protected int procedure(AbstractNField dfhcommarea) throws ContextException {
        WS.len.setValue(WS.tcpSockParm.length());
        supernaut.retrieve()
                .into(WS.tcpSockParm)
                .length(WS.len)
                .nohandle()
                .exec();
        try {
            var socket = EzaSocket.takesocket(WS.tcpSockParm.giveTakeSocket.getInt());
            socket.getSocket().getOutputStream().write("MESSAGE FROM EZASERVER".getBytes());
            socket.close();
        } catch (IOException e) {
            throw new AbendException("Error while taking socket: " + e.getMessage());
        }
        return RETURN_CODE;
    }
}
